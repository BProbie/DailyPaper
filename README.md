# 场景描述

## 我刚重置完电脑看着电脑桌面上展示的操作系统自带的经典壁纸陷入了沉思：我该换一张怎样的壁纸呢？

## 这张壁纸我好像已经用了好久了早就看腻了因此我再一次陷入了沉思：我又该换一幅怎么样的新壁纸呢？

## 今天是新年，生活中周围都洋溢着新春喜庆的大红，看着电脑上不和氛围的壁纸，但是在浏览器上又找不到可以令我满意的壁纸，既懒得花大量时间找合适的壁纸，又宅在家里死敲代码的我再再一次陷入了沉思：我可不可以让符合此时此刻我心情的壁纸一张一张轮流地自己跑到我的电脑桌面上呢？

## 因此，DailyPaper(Daily+AI+Wallpaper)，诞生了！



# 软件简介

## 支每隔自定义时间推送高清个性壁纸。

## 支持以用户喜好为依据创作生成、推荐推送每日壁纸。

## 支持上传素材图像解析并作为每日壁纸的推送依据。

## 支持用户手动调整各项软件参数让用户体验更符合用户习惯。

## 支持用户手动调整各种软化设置和软件高级设置

# 使用教程

## Chat聊天（AI智能聊天）

### 你可以在这里与本软件的AIAgent小Day畅聊你的想法，或上传素材，并让它帮你创作、生成、管理壁纸。按<u>`【Ctrl+Enter】`</u>组合键发送消息。在聊天界面再次点击聊天按钮可开启新的聊天。

## Live动图（自定义壁纸）

### 你可以在这里导入高清图片设为壁纸，也可以导入GIF动图作为高清动态壁纸。

## Daily每日（推荐每日壁纸）【作者主推功能】

### 你可以在这里开启每日推送壁纸，描述或上传壁纸偏好，设置推送壁纸的时间间隔，开机一换，启动一换，一日一换，亦或几小时一换，几分钟一换。小Day会基于你的喜好结合今天的时间地点、节日节气、历史文化等等各项因素向你展示具有浓郁气氛的精美高清壁纸！

## Param参数（配置参数）

### 你可以在这里修改软件的各项参数，让其更符合你的个人使用习惯。

## Setting设置（配置设置）

### 你可以在这里调配各种软件设置，甚至接入个人插件。

## Renew更新（更新）

### 你可以在这里，一键点击，放手更新。



# 技术细节

## Chat

### 技术点在于这里维护一个AIAgent小Day，主要分成三大模块。一，Information信息背景，为它提供了软件背景和现实数据，让它的回复和创造更符合生活实际。二，Memory记忆，为它编写了一套记忆体系，以便让它可以结合长上下文分析数据，给出更具有个性化的回答和创作。三，FunctionCalling（Skills）技能，这里主要维护了几套工具调用分别是文本生成文本、文本生成图片、图片生成文本的识别和调用。

## Live

### 技术点在于对于对动态图片GIF的分片切片，以及对于切片细节的修正，参数的修复。同时对于切片数据在本地的保存、离散复用和短期维护、扩展。还有对于Native的利用，调用操作系统自带的动态链接库（Windows-64对应的是user.dll）来进行壁纸的更换和动态壁纸的实现及优化维护。

## Daily

### 技术点在于对推送壁纸线程的维护，AI软编程的实现，推送提示词的优化，用户图像喜好的结合，系统级开机自启的实现，权限的灵活管理，程序路径及线程的维护细节。

## Param

### 技术点在于对数据的灵活的抽拉，存储，使用以及渲染，兼具对功能的动态调整，以及对于重置功能的实现和复用。

## Setting

### 技术点在于对类似vue的数据渲染UI响应式编程的理解和实现，采用约定大于配置的思想架构，自动记忆参数的实现，默认参数和用户参数之间的协调平衡。调试模式的实现与日志功能的适配，以及对插件补丁的接入和利用的实现模式。

## Renew

### 技术点在于对网络的监测，资源占用的解决和优化，自动更新的实现，数据同步的尝试，以及在网络加速环境下访问web资源时对于SSL及其相关安全证书问题的安全评估和信任操作。
