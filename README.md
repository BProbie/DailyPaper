# 使用教程

## Chat（AI智能聊天）

### 你可以在这里与本软件的Agent小Day畅聊你的想法让它帮忙生成、设置壁纸，或下载图片。按`Ctrl+Enter`组合键发送消息。

## Live（自定义壁纸）

### 你可以在这里导入高清图片设为壁纸，也可以导入GIF动图设置动态壁纸。

## Daily（每日壁纸）

### 你可以在这里设置每日推送的壁纸风格，一日一换，或几分钟一换。

## Hobby（个性爱好）

### 修改你的个人爱好，依此让生成的壁纸更贴合你的个性。

## Setting（参数设置）

### 你可以在这里调配各种参数，甚至接入个人插件。

## Renew（更新）

### 一键点击，放手更新。



# 技术细节

## Chat

### 技术点在于这里维护了一个Agent小Day，分三大块。一，Information信息背景，为它提供了软件背景和现实数据。二，Memory记忆，为它编写了一套记忆体系，以便让它可以结合上下文分析数据，给出个性化的回答。三，FunctionCalling（Skills）技能，这里主要维护了两套工具调用分别是文本生成和图片生成。其余的是，参数的整合，UI的排版等。

## Live

### 技术点在于GIF切片，本地数据的维护扩展，以及利用Native调用操作系统自带的动态链接库（User.dll）来进行更换壁纸的操作。

## Daily

### 技术点在于操作系统系统级开机自启，需要灵活的权限管理，路径搜寻，线程维护，网络检测。

## Hobby

### 技术点在于软编程的尝试，AI自己创作代替硬编码。

## Setting

### 技术点在于类似vue的数据渲染UI，自动参数，默认参数和用户参数之间的协调平衡。

## Renew

### 技术点在于网络检测，资源占用，自动更新等。
