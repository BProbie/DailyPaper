# 使用教程

## Chat聊天（AI智能聊天）

### 你可以在这里与本软件的AIAgent小Day畅聊你的想法，或上传素材，并让它帮你创作、生成、管理壁纸。按`【Ctrl+Enter】`组合键发送消息。

## Live动图（自定义壁纸）

### 你可以在这里导入高清图片设为壁纸，也可以导入GIF动图作为高清动态壁纸。

## Daily每日（推荐每日壁纸）

### 你可以在这里开启每日推送壁纸，描述或上传壁纸偏好，设置推送壁纸的时间间隔，开机一换，启动一换，一日一换，亦或几小时一换，几分钟一换。

## Param参数（配置参数）

### 你可以在这里修改软件的各项参数，让其更符合你的个人使用习惯。

## Setting设置（配置设置）

### 你可以在这里调配各种软件设置，甚至接入个人插件。

## Renew更新（更新）

### 你可以在这里，一键点击，放手更新。



# 技术细节

## Chat

### 技术点在于这里维护一个AIAgent小Day，主要分成三大模块。一，Information信息背景，为它提供了软件背景和现实数据，让它的回复和创造更符合生活实际。二，Memory记忆，为它编写了一套记忆体系，以便让它可以结合长上下文分析数据，给出更具有个性化的回答和创作。三，FunctionCalling（Skills）技能，这里主要维护了几套工具调用分别是文本生成文本、文本生成图片、图片生成文本的识别和调用。

## Live

### 技术点在于对于对动态图片GIF的分片切片，以及对于切片细节的修正，参数的修复。同时对于切片数据在本地的保存、离散复用和短期维护、扩展。还有对于Native的利用，调用操作系统自带的动态链接库（Windows-64对应的是user.dll）来进行壁纸的更换和动态壁纸的实现及优化维护。

## Daily

### 技术点在于对推送壁纸线程的维护，AI软编程的实现，推送提示词的优化，用户图像喜好的结合，系统级开机自启的实现，权限的灵活管理，程序路径及线程的维护细节。

## Param

### 技术点在于对数据的灵活存储，使用和渲染，兼具对功能的动态调整。

## Setting

### 技术点在于对类似vue的数据渲染UI的理解和实现，自动记忆参数的实现，默认参数和用户参数之间的协调平衡。以及对插件补丁的接入和利用的实现模式。

## Renew

### 技术点在于对网络的监测，资源占用的调整和优化，自动更新的实现，数据同步的模型。
